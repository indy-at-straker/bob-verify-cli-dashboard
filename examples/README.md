# Example Files

This directory contains sample files for testing the Straker Verify Dashboard.

## Files

### sample_en.txt
A simple English text file demonstrating basic translation quality evaluation features.

## Usage

You can use these files to test the dashboard functionality:

1. Launch the dashboard: `python -m src.main`
2. Press `u` to upload a file
3. Select one of the example files
4. View the quality evaluation results

## Creating Your Own Test Files

You can add your own test files in various formats:
- `.txt` - Plain text files
- `.docx` - Microsoft Word documents
- `.pdf` - PDF documents
- `.html` - HTML files

The mock API client will simulate processing and return quality scores for demonstration purposes.
